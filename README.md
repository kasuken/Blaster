# Blaster

A state container for Blazor
## Tech Stack

Blazor, C# 10, .NET 6, WebAssembly

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/kasuken/blaster
```

Go to the project directory

```bash
  cd blaster
```

```bash
  dotnet run
```
